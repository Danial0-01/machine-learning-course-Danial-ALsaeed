This Lab applies Machine Learning to predict the yearly spending of E-commerce customers using a Linear Regression model. By analyzing user engagement metrics like time spent on the mobile app versus the website, the model mathematically identifies which platform drives more revenue. The workflow includes data preparation, model training, and performance evaluation using standard regression metrics (MAE, RMSE, R^2). Ultimately, the insights extracted from the model help the company make a data-driven decision on whether to focus their engineering efforts on their app or their website.
